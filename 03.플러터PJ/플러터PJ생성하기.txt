[ 플러터 PJ 생성하기 ]

-> 플러터 명령어를 통해 개발표준 팩키지를 설치하여 
개발을 더 쉽고 용이하게 할 수 있도록 구글이 지원함

-> 참고 : 표준 개발 프로젝트 팩키지를 보일러플레이트
라고 한다!(boilerplate-개발 프로젝트 표준 코드. 전체 환경을 만들어줌)

[ 설치 순서 ]
1. 설치할 위치로 이동한다(cd 명령어 사용)

2. cmd명령어 창에서 프로젝트 생성 플러터 명령어를 실행함
->>>>>> flutter create 프로젝트이름

((프로젝트 명명법))
-> 숫자로 시작불가, - 등 특수문자 사용불가
-> 일반적으로 언더바를 사용한 단어연결로 명명함!
예) app_01_store
->> flutter create app_01_store

3. 설치 후 프로젝트 파일로 이동한다(cd 명령어로 이동)

4. 프로젝트 폴더 위치(\app_01_store)에서 모듈을 실행시킨다
->>> flutter run
-> 옵셥 중 2번 크롬브라우저 선택 (2라고 입력하면 모듈이 자동적으로 실행됨!)
-> 모듈 실행하여 일정시간 후 브라우저로 로딩됨!!!!
(보기 편하게 모바일버전으로 보이도록 개발자도구에서 디바이스를 바꿔주면 보기 좋음)
참고) 모듈 실행 후 cmd 창에 r키를 누르면 자동 핫리로드로 재실행 새로고침해줌!

5. 모듈 종료방법 :  
Cmd창에서 ctrl+C 하면 닫겠냐고 물어봄 . 이때 y 입력하면 닫음(리액트 닫기랑 동일함)
또는 그냥 터미널 쓰레기통 아이콘 클릭, 또는 브라우저 닫기로도 닫기 가능함!
